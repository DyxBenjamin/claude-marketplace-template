# Claude Marketplace

## Adding a New Plugin

Create only:

```text
plugins/<name>/.claude-plugin/plugin.json
```

Everything else is automated:

- `marketplace.json` - synced by CI
- `release-please-config.json` - generated by CI
- Version bumps - handled by Release Please

## plugin.json Template

```json
{
  "name": "plugin-name",
  "version": "0.1.0",
  "description": "What this plugin does",
  "author": {
    "name": "CHANGE_ME",
    "email": "CHANGE_ME@example.com"
  },
  "keywords": ["relevant", "tags"]
}
```

## Commit Convention

Use conventional commits for automatic versioning:

- `feat:` → minor bump
- `fix:` → patch bump
- `feat!:` or `BREAKING CHANGE:` → major bump
